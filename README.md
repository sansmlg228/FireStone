## üìñ About The Project

**Fire Stone Mod** is a long-term Minecraft modification project developed for the **Fire Stone** story-driven server. This mod provides players with deep tools for character development and lore customization, allowing everyone to create and evolve their unique story within the server's universe.

The development of this mod is ongoing and will continue for a long time, with regular updates adding new features and expanding roleplaying possibilities.

## ‚ú® Key Features

*   **Dynamic Character Lore:** Every player can change and develop their character's backstory and personality.
*   **Integrated Lore System:** Special interfaces and mechanics for convenient lore management.
*   **Server-Specific Content:** The mod is tailored specifically for the Fire Stone server and its storyline.
*   **Regular Updates:** New features and improvements will be added continuously.
